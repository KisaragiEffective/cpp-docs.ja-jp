# <a name="dq"></a>DQ

*初期化子*ごとに8バイトのストレージを割り当て、必要に応じて初期化します。 型が有効である任意の場所で、型指定子として使用することもできます。 **DQ**は、 [QWORD](../../assembler/masm/qword.md)のシノニムです。

## <a name="syntax"></a>構文

> ⟦*name*⟧ **DQ** *初期化子*⟦ __、__ *初期化子*...⟧
