# <a name="operator-sectionrel"></a>SECTIONREL 演算子

最終的な実行可能ファイルのターゲットを含むセクションを基準とした、式の相対オフセットセクションを返します。

## <a name="syntax"></a>構文

> **Sectionrel** *式*

## <a name="remarks"></a>コメント

**Sectionrel**は、COFF オブジェクトの出力でのみ使用できます。
