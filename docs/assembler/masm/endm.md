# <a name="endm"></a>ENDM

マクロまたは繰り返しブロックを終了します。

## <a name="syntax"></a>構文

> **ENDM**

## <a name="remarks"></a>コメント

「[マクロ](../../assembler/masm/macro.md)、 [for](../../assembler/masm/for-masm.md)、 [forc](../../assembler/masm/forc.md)、 [REPEAT](../../assembler/masm/repeat.md)、または[WHILE](while-masm.md)」を参照してください。
