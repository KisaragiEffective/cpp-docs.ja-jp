# <a name="popcontext"></a>POPCONTEXT

現在の*コンテキスト*の一部またはすべてを復元します ( [pushcontext](../../assembler/masm/pushcontext.md)ディレクティブによって保存されます)。 *コンテキスト*は、(32-bit MASSM only)、 **RADIX**、**リスティング**、 **CPU** (32 ビット MASSM のみ)、または**すべて**を**想定**できます。

## <a name="syntax"></a>構文

> **Popcontext** *コンテキスト*
