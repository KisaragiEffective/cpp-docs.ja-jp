# <a name="ml-fatal-error-a1009"></a>ML の致命的なエラー A1009

**行が長すぎます**

ソースファイル内の行が512文字の制限を超えています。

複数の物理的な行が行連結文字 (\) と連結されている場合、結果として得られる論理行は引き続き512文字に制限されます。
