# <a name="df"></a>DF

は、*初期化子*ごとに6バイトのストレージを割り当て、必要に応じて初期化します。 **DF**は[fword](../../assembler/masm/fword.md)のシノニムです。

## <a name="syntax"></a>構文

> ⟦*name*⟧ **DF** *initializer* ⟦ __、__ *initializer* ...⟧

## <a name="remarks"></a>コメント

型が有効である任意の場所で、型指定子として使用することもできます。
