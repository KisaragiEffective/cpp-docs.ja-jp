# <a name="real10"></a>REAL10

各*初期化子*に対して、10バイトの浮動小数点数を割り当てて、必要に応じて初期化します。

## <a name="syntax"></a>構文

> *名前* **REAL10** *initializer* ⟦ __、__ *初期化子*...⟧
