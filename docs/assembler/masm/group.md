# <a name="group-32-bit-masm"></a>グループ (32 ビット MASM)

指定された*セグメント*を*名前*と呼ばれるグループに追加します。 (32 ビット MASM のみ。)

## <a name="syntax"></a>構文

> 名前グループセグメント [[, セグメント]]...
