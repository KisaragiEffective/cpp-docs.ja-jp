# <a name="operator-ptr"></a>PTR 演算子

最初の演算子は、指定された*型*を持つ*式*を強制的に処理します。 2番目の演算子は、*型*へのポインターを指定します。

## <a name="syntax"></a>構文

> **PTR** *式*の入力
>
> \[*distance*] **PTR** *型*
