# <a name="ml-nonfatal-error-a2189"></a>ML の致命的でないエラー A2189

**セグメントアラインメントとの組み合わせが無効です**

**ALIGN**または `EVEN` ディレクティブによって指定されたアラインメントが、 **segment**ディレクティブで指定された現在のセグメントのアラインメントを超えています。
