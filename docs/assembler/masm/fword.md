# <a name="fword"></a>FWORD

は、*初期化子*ごとに6バイトのストレージを割り当て、必要に応じて初期化します。

## <a name="syntax"></a>構文

> ⟦*name*⟧ **fword** *初期化子*⟦ __、__ *初期化子*...⟧

## <a name="remarks"></a>コメント

型が有効である任意の場所で、型指定子として使用することもできます。
