# <a name="sqword"></a>SQWORD

*初期化子*ごとに8個の符号付きバイトのストレージを割り当て、必要に応じて初期化します。 型が有効である任意の場所で、型指定子として使用することもできます。

## <a name="syntax"></a>構文

> **sqword** *初期化子*__の名前を⟦、__ *initializer* ...⟧
