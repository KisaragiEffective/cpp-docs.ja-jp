# <a name="586p-32-bit-masm"></a>. 586P (32 ビット MASM)

Pentium プロセッサのすべての命令 (特権を含む) のアセンブリを有効にします。 (32 ビット MASM のみ。)

## <a name="syntax"></a>構文

> .586P
