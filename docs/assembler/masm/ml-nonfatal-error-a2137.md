# <a name="ml-nonfatal-error-a2137"></a>ML の致命的でないエラー A2137

**呼び出す引数が少なすぎます**

[INVOKE](../../assembler/masm/invoke.md)ディレクティブを使用して渡された引数の数が、呼び出されているプロシージャのプロトタイプで指定されている必須パラメーターの数よりも短くなっています。
