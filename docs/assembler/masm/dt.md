# <a name="dt"></a>DT

*初期化子*ごとに10バイトのストレージを割り当て、必要に応じて初期化します。 型が有効である任意の場所で、型指定子として使用することもできます。 **DT**は、 [t](../../assembler/masm/tbyte.md)のシノニムです。

## <a name="syntax"></a>構文

> ⟦*name*⟧ **DT** *initializer* ⟦ __,__ *initializer* ...⟧
