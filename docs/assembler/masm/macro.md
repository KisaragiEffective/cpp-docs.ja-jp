# <a name="macro"></a>MACRO

マクロブロックを*名前*としてマークし、マクロが呼び出されたときに渡される引数の*パラメーター*のプレースホルダーを確立します。

## <a name="syntax"></a>構文

> *name***マクロ**⟦*parameter* ⟦ **: REQ** |: =*既定* |  **: VARARG**⟧...⟧\
> *ステートメント*の\
> **Endm** ⟦*value*⟧

## <a name="remarks"></a>コメント

マクロ関数は、呼び出し元のステートメントに*値*を返します。
