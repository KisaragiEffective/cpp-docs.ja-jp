# <a name="ml-nonfatal-error-a2078"></a>ML の致命的でないエラー A2078

**命令では、間接的なアドレス指定は許可されていません**

条件付きジャンプまたはループは、メモリオペランドを受け取ることができません。 相対アドレスまたはラベルを指定する必要があります。
